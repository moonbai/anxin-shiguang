## 1. 产品概述

「安心食光」是一款以孕期 / 月子期饮食禁忌为核心安全标准的菜谱应用。菜谱内容直接采用开源项目 [HowToCook](https://github.com/Anduin2017/HowToCook) 的菜谱数据，在其之上叠加一层"食材—孕产阶段—安全等级"的禁忌判定：用户先选择所处阶段（孕期 / 月子期），应用即按安全等级过滤与排序菜谱，并标记每一道菜对当前阶段的适宜性。遇到想学的菜，可一键跳转抖音或小红书搜索视频教程。

- 解决问题：孕期 / 月子期饮食禁忌信息分散在零散文章里，新手妈妈看完菜谱仍不知道"这菜我现在能不能吃、哪些料要替换"。
- 目标用户：备孕、孕期、月子期女性及其家属；追求"吃得安心、做得明白、看得见教程"的精细化人群。
- 核心价值：把"禁忌"从被动查询变为主动前置提醒——打开菜谱第一眼先看到安全结论。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 访客 | 直接访问 | 浏览菜谱、查看禁忌、跳转搜索 |
| 定制用户 | 选择阶段(孕期/月子) | 按安全等级过滤菜谱、个性化禁忌提醒 |

### 2.2 功能模块
1. **首页（安心菜谱大厅）**：阶段切换器、今日安心推荐、禁忌速查入口、按安全等级过滤的菜谱墙、分类导航
2. **菜谱详情页**：安全结论卡（首要）、食材逐项禁忌标注、HowToCook 原始做法、抖音 / 小红书视频跳转
3. **禁忌速查页**：孕期 / 月子期禁忌食材全库、安全等级色块、点击查看原因与替代建议

## 3. 核心流程

用户打开应用 → 首页选择所处阶段（孕期 / 月子期） → 首页自动按安全等级过滤并置顶推荐适宜菜谱 → 浏览或搜索菜谱 → 进入菜谱详情页，第一眼看到安全结论 → 查看食材逐项禁忌标注与替代建议 → 查看做法步骤 → 点击"抖音 / 小红书"按钮跳转观看视频教程。亦可从首页直接进入禁忌速查页，按阶段浏览全部禁忌食材。

## 4. 界面设计

### 4.1 设计风格
- **主题方向**：暖意有机风（Organic & Nurturing）—— 像一本设计精良的母婴生活杂志，温暖、安心、可信；安全信息用克制而醒目的色块强调。
- **主色**：暖陶土橙 `#C56B3F`（品牌主色，安心感）+ 鼠尾草绿 `#7C8B6A`（次色，自然食材）
- **辅色**：奶油米 `#F7F1E8`（背景）+ 深咖文字 `#3A2E26`
- **安全徽章色**：适宜=鼠尾草绿 `#7C8B6A`，慎食=琥珀黄 `#D4A24C`，禁食=砖红 `#B5503E`
- **按钮风格**：圆角胶囊按钮，主按钮陶土橙填充，次按钮描边；视频跳转按钮用抖音黑 `#161823`、小红书红 `#FF2442`
- **字体**：标题用衬线字体（Noto Serif SC / Cormorant Garamond）显文气；正文用 Noto Sans SC
- **布局**：卡片式网格 + 杂志感留白，顶部 sticky 导航；安全结论卡在详情页用强调色块置顶
- **图标**：线性 SVG 图标（lucide-react），保持质感，极少使用 emoji

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页 | 阶段切换器 | 顶部胶囊切换，选中态橙色填充，背景奶油米 |
| 首页 | 今日安心推荐 | 三张并排卡片，绿色"适宜"徽章醒目，配大图 |
| 首页 | 禁忌速查入口卡 | 大色块卡片，左侧图标右侧文字，柔和投影 |
| 首页 | 菜谱卡片墙 | 2-3 列响应式网格，安全徽章左上角优先，卡片圆角+柔和投影 |
| 菜谱详情页 | 安全结论卡 | 顶部全宽带色块，大字结论 + 一句话说明，按等级变色 |
| 菜谱详情页 | 头图区 | 全宽大图，菜名衬线大字叠加，HowToCook 来源标签 |
| 菜谱详情页 | 食材禁忌清单 | 列表每行：圆点色标 + 食材名 + 等级徽章，禁忌项可展开 |
| 菜谱详情页 | 视频跳转按钮 | 两个并排胶囊按钮，抖音黑、小红书红，带搜索图标 |
| 禁忌速查页 | 食材卡片 | 左侧色条标识等级，右侧名称+原因，点击展开 |

### 4.3 响应式
- 桌面优先设计，最大宽度 1200px 居中
- 平板：菜谱网格 2 列，今日推荐 2 列
- 移动端：单列堆叠，导航变汉堡菜单，按钮触控友好（≥44px），视频按钮并排不变窄

不适用。
